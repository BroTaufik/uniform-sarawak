# Audit Penerimaan Kain Pakaian Seragam TABIKA KEMAS Sarawak 2026

Dashboard HTML interaktif untuk audit penerimaan kain pakaian seragam kanak-kanak TABIKA KEMAS Sarawak.

## Maklumat

- **Organisasi:** KEMAS Negeri Sarawak
- **Skop:** 31 Parlimen (P.192 - P.222)
- **Fasa:** Fasa 1 - Fasa 4
- **Kemaskini:** 22 Jun 2026
- **Klasifikasi:** SULIT - Penggunaan Dalaman Sahaja

## Kandungan Dashboard

| Bahagian | Keterangan |
|---|---|
| Ringkasan Eksekutif | 7 KPI cards + 6 carta overview |
| Status Kesempurnaan | Taburan lengkap/tidak lengkap per Parlimen |
| TK-1 Kualiti Fabrik | 3 audit findings |
| TK-2 Kesempurnaan Set | 3 audit findings |
| TK-3 Aksesori | 4 audit findings |
| TK-4 Kawalan Vendor | 3 audit findings |
| Status Parlimen | Bar chart 31 Parlimen |
| Semua Findings | 12 findings dengan filter severity |
| Cadangan Tindakan | 10 cadangan bertanda masa |
| Repo Aduan | 16 folder daerah (Google Drive) |
| Repo Dokumen | 3 dokumen embedded |

## Teknologi

- Single-file HTML (self-contained)
- Chart.js 4.4.1
- Glassmorphism + Skeuomorphism UI
- Dark/Light mode toggle
- Dokumen embedded (base64)
